# mood-tracker-java
Java OOP-based Mood Tracker application with GUI for tracking daily emotions

I. Project Overview
Mood Tracker is a simple **Java desktop application** designed to help users record and monitor their daily emotions and activities.This application aims to increase **self-awareness** by allowing users to reflect on their mood patterns and understand how daily activities influence their emotional state.

---

II. OBJECTIVES
* Help users track their daily mood
* Analyze the relationship between mood and activities
* Encourage self-reflection and emotional awareness
* Provide simple suggestions based on mood

---

III. FEATURES

1. **Mood Tracking**
  Users can select their mood (happy, sad, angry, anxious, tired)

2. **Activity Logging**
  Users can record daily activities related to their mood

3. **Mood Suggestions**
  The system provides simple suggestions based on mood

4.  **Graphical User Interface (GUI)**
  Easy-to-use interface for better user experience

---

VI. How to Run 

1. Clone this repository:
```bash id="tv5d2c"
git clone https://github.com/username/mood-tracker-java.git
```
2. Compile the program:
```bash id="t93t0b"
javac *.java
```
3. Run the application:
```bash id="z8k8zw"
java MoodTrackerGUI
```
